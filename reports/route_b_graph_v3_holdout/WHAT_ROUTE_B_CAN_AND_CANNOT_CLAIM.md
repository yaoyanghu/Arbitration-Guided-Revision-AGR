# What Route B Can And Cannot Claim

## What Route B Can Claim

1. Route B is not an empty shell.
The prototype builds interpretable local relation structures with `support`, `contradict`, `update`, and `corroborate` edges.

2. Route B can improve over Route A alone on the hardened and held-out subsets.
The current graph aggregation beats `A only` and also beats `update_only`.

3. Route B is useful as a conflict-focused analysis layer.
The casebooks and stratified reports show that Route B helps surface where temporal conflict and source conflict matter, especially in mixed hard cases.

## What Route B Cannot Claim

1. Route B has not proven independent graph value.
On the held-out hard subset, `full_graph = case_aware_non_graph`. The current gains can be reproduced by a matched non-graph aggregator using the same information.

2. Route B has not earned a Route B main experiment.
Because the matched non-graph control closes the gap, the current evidence does not justify a standalone graph-method line.

3. Route B has not established a general graph retrieval claim.
Current gains are still narrow, controlled, and conflict-focused. They do not support a broader claim that graph structure itself is the key source of improvement.

4. Route B does not imply anything about Route C.
No Route C capability follows from the current Route B checkpoint.

## Why Route B Cannot Enter Main Experiment

- The main blocking fact is not low performance; it is missing causal attribution.
- A method-line Route B would require evidence that graph structure adds something beyond matched non-graph aggregation.
- The held-out checkpoint does not show that.

## Why Route B Fits As An Analysis Module For Now

- It provides interpretable structure over conflict cases.
- It helps organize update / contradiction / corroboration patterns.
- It remains useful for diagnosis even when it is not yet a uniquely necessary method.
- This is a productive and honest resting position until a truly graph-native gain is shown.
